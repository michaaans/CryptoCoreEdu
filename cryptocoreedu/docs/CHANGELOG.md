# CHANGELOG

## (Milestone 1: Project Foundation and Basic Blocks)
### Добавлено
- Создан репозиторий проекта CryptoCore с базовой структурой.
- Добавлены README.md и система сборки (Makefile для C / setup.py для Python).
- Реализована поддержка CLI с аргументами:
  - `--algorithm aes`
  - `--mode ecb`
  - `--encrypt` / `--decrypt`
  - `--key <key>`
  - `--input <file>`
- Реализованы AES-128 шифрование и дешифрование в режиме ECB (с использованием библиотеки для AES-раундов).
- Добавлена базовая работа с файлами (чтение/запись).
- Начата работа над тестированием: проверка шифрования-дешифрования для ECB.

### Изменено
- Оптимизирована структура кода для удобства расширения.
- Улучшена обработка ошибок ввода CLI.

### Исправлено
- Незначительные баги при чтении файлов с нестандартными кодировками.

## В разработке (Milestone 2: Modes of Operation and IV Handling)
### Планируется
- Добавление поддержки режимов шифрования: `--mode cbc`, `--mode cfb`, `--mode ofb`, `--mode ctr`.
- Реализация каждого режима (CBC, CFB, OFB, CTR) с нуля.
- Добавление генерации безопасного случайного IV при шифровании.
- Реализация обработки IV при дешифровании (чтение из начала файла).
- Интеграция с OpenSSL для кросс-тестирования каждой моды.

## В разработке (Milestone 3: Cryptographically Secure Random Number Generation)
### Планируется
- Реализация модуля CSPRNG в библиотеке.
- Добавление функции `generate_random_bytes(num_bytes)`.
- Интеграция генерации случайного 16-байтного ключа, если `--key` не указан.
- Автоматическая генерация случайного IV для соответствующих режимов.
- Добавление статистических тестов для 1000 ключей (проверка на уникальность).

## В разработке (Milestone 4: Hashing and Data Integrity)
### Планируется
- Добавление новой CLI команды: `cryptocore dgst --algorithm sha256 --input <file>`.
- Реализация SHA-256 с нуля (образовательная задача).
- Поддержка минимум двух алгоритмов хеширования (SHA-256, SHA3-256 или BLAKE2).
- Тестирование: сравнение с результатами `sha256sum`.

---

## В разработке (Milestone 5: Message Authentication Codes)
### Планируется
- Расширение команды `dgst` поддержкой `--hmac --key <key>`.
- Реализация HMAC с нуля на основе SHA-256.
- Бонус: реализация AES-CMAC.
- Тестирование HMAC: проверка целостности и аутентичности данных.

---

## В разработке (Milestone 6: Authenticated Encryption)
### Планируется
- Реализация Encrypt-then-MAC как составной функции.
- Реализация режима GCM с нуля (включая умножение в поле Галуа).
- Добавление новых флагов: `--mode gcm` и `--aad <data>`.
- Тестирование аутентифицированного шифрования с AAD.

## В разработке (Milestone 7: Key Derivation and Protocol Foundation)
### Планируется
- Добавление новой CLI команды: `cryptocore derive --password <pass> --salt <salt> --iterations 1000000 --length 32`.
- Реализация PBKDF2-HMAC-SHA256 с нуля.
- Добавление функции для иерархического получения ключей из мастер-ключа.
- Тестирование: сравнение с `openssl kdf`.
